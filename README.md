# Calculator

Language: Python
Version: 2.3 stable.
Command-line calculator. 
Contains instruction on how to use the program,
Has built-in commands like: /help, /history, etc.
Has random example generation.
Includes both Pemdas and Letori evaluation (which can be toggled at will).
Things that will (hopefully) be added in next version (3.0): Parenthesis, Memory, and Variable support and the ability to add custom functions.
